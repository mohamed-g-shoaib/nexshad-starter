## [0.1.0] - 2025-12-13

### Added

- Initial release of Nexcn starter boilerplate
- Interactive CLI tool - Run `npx nexcn my-app` to create new projects
- Next.js 16 with App Router and React 19
- TypeScript with strict mode enabled
- Tailwind CSS v4 with Base UI components
- shadcn/ui integration for customizable components
- next-intl for English/Arabic internationalization with RTL support
- Vitest for unit testing
- Playwright for E2E testing
- ESLint and Prettier for code quality
- Husky pre-commit hooks with lint-staged
- SEO optimization with metadata management
- Comprehensive documentation in `/docs` folder
- Example components and pages
- Dark mode support with next-themes
- Responsive design system

### CLI Features

- Beautiful ASCII art logo
- Automatic prerequisite checking
- Interactive project setup
- Colorful terminal output with progress indicators
- Automatic dependency installation
- Git repository initialization
